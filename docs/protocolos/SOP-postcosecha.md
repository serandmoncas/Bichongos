# SOP — Postcosecha: Procesamiento, Conservación y Empaque

**Aplica a:** Todas las especies
**Nivel:** Principiante
**Tiempo:** Variable según volumen

---

## Flujo de postcosecha

```
Cosecha en cápsula
      ↓
Limpieza y clasificación (Zona C)
      ↓
       ┌──────────────┬──────────────────────┐
       ▼              ▼                      ▼
  Consumo      Conservación          Procesamiento
  inmediato    en fresco              para extracto
  (24–48 h)   (refrigeración)        (secado / polvo)
```

---

## 1. Cosecha

### Momento óptimo de cosecha

| Especie | Indicador de cosecha |
|---------|---------------------|
| Orellana | Bordes del sombrero aún enrollados hacia abajo; 5–15 cm diámetro |
| Shiitake | Velo parcial sin romper o recién abierto; 5–10 cm |
| Portobello | Velo abierto, láminas visibles, 10–15 cm (o cerrado para Paris/Crimini) |
| Melena de León | Pompón 8–15 cm, color blanco, sin amarillamiento |
| Enoki | Tallos 10–15 cm, sombreros < 1 cm |
| Reishi | Borde blanco recién desaparecido, superficie roja uniforme |
| Cordyceps | Estróbilos 5–10 cm, superficie lisa (antes de liberar esporas) |

### Técnica de cosecha

- **Orellana, Shiitake, Melena:** torcer y jalar suavemente en la base. No usar cuchillo (riesgo bacteriano).
- **Portobello, Crimini:** cortar limpio con cuchillo desinfectado.
- **Reishi, Cola de Pavo, Cordyceps:** cortar con tijeras desinfectadas.
- **Nunca dejar hongos sobremaduros** en la cápsula — liberan esporas y pueden contaminar.

---

## 2. Limpieza y clasificación

1. **Limpiar con brocha suave** — no lavar con agua (acelera la degradación).
2. **Clasificar por tamaño** y calidad:
   - **Categoría A:** tamaño uniforme, sin daños → restaurantes / mercado premium
   - **Categoría B:** tamaño variable, leves imperfecciones → mercado general / plazas
   - **Categoría C:** fragmentos, rotos, sobremaduros → deshidratación / extracto
3. **Pesar cada categoría.** Registrar en el lote.

---

## 3. Conservación en fresco

| Variable | Recomendación |
|---------|--------------|
| Temperatura | 2–4 °C (nevera estándar) |
| Empaque | Bandeja de espuma + film perforado, o bolsa de papel |
| Evitar | Bolsas herméticas sin perforaciones (acumulan CO₂ y pudrición) |

### Vida útil en fresco

| Especie | Días en fresco |
|---------|---------------|
| Orellana | 5–10 días |
| Shiitake | 10–15 días |
| Portobello | 7–12 días |
| Melena de León | 7–10 días |
| Enoki | 10–14 días |
| Reishi | No aplica — se procesa seco |

---

## 4. Deshidratación

### Parámetros del deshidratador

| Especie | Temperatura | Tiempo | Humedad residual objetivo |
|---------|------------|--------|--------------------------|
| Orellana | 50–55 °C | 6–10 h | < 8 % |
| Shiitake | 55–60 °C | 8–12 h | < 8 % |
| Melena de León | 50 °C | 8–12 h | < 8 % |
| Reishi | 55–60 °C | 10–16 h | < 8 % |
| Cordyceps | 45–50 °C | 8–12 h | < 8 % |

**Prueba de humedad residual:** el producto seco debe quebrarse limpiamente al doblarlo. Si dobla sin quebrar, aún tiene humedad.

### Rendimiento seco/húmedo (referencia)

| Especie | Ratio húmedo → seco |
|---------|-------------------|
| Orellana | 10:1 (10 kg frescos → 1 kg seco) |
| Shiitake | 8:1 |
| Reishi | 5:1 (ya es seco) |
| Cordyceps | 6:1 |

---

## 5. Producción de polvo

1. Verificar que el producto seco tiene < 8 % de humedad.
2. Moler en licuadora o molino de especias.
3. Tamizar (malla 100 mesh para polvo fino).
4. Almacenar en frasco hermético ámbar con desecante (sílica gel).

## 6. Empaque y etiquetado

### Etiqueta obligatoria para venta

```
┌─────────────────────────────────┐
│  BICHONGOS                      │
│  [Nombre del hongo]             │
│                                 │
│  Peso neto: _____ g             │
│  Fecha cosecha: ___________     │
│  Lote: ___________              │
│  Conservar a 2–4 °C             │
│  Consumir antes de: ___________ │
└─────────────────────────────────┘
```

### Tipos de empaque

| Formato | Empaque | Para |
|---------|---------|------|
| Fresco categoría A | Bandeja espuma + film | Restaurantes |
| Fresco categoría B | Bolsa papel kraft perforada | Plazas / directo |
| Seco en trozo | Bolsa kraft sellada al vacío | Tiendas naturistas |
| Polvo | Frasco ámbar + tapa inviolable | Suplementos |
| Cápsulas | Blister o frasco oscuro 60 u | Farmacia / naturistas |
